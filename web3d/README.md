# 北京交通大学就餐虚拟仿真系统 Web3D 版

这是在原 C++ 动画程序之外新增的网页端 3D 虚拟仿真版本，使用 Three.js 构建，不依赖 npm 安装即可运行。

## 运行方式

在 `web3d` 目录启动本地静态服务器：

```powershell
cd D:\code\软件综合实训\web3d
python -m http.server 5173 --bind 127.0.0.1
```

然后在浏览器打开：

```text
http://127.0.0.1:5173/
```

## 功能

- 3D 食堂空间：入口、出口、取餐窗口、等座区、休息区、A/B/C 就餐区。
- 实时学生流：进入、排队、取餐、选座、就餐、休息和离开。
- 交互相机：鼠标拖拽旋转，滚轮缩放，右键平移。
- 实时监控：系统人数、排队人数、座位占用率、完成取餐人数。
- 可视化图表：趋势图、窗口负载柱状图、学生状态分布图。
- 控制项：暂停/继续、重置、俯视/透视切换、速度调节、热力区域开关。
